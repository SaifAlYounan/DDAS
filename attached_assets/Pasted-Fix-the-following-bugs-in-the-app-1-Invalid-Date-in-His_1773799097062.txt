Fix the following bugs in the app:

1. "Invalid Date" in History — When analyses are saved to history, the dates show as "Invalid Date Invalid Date". The date is not being stored or parsed correctly when saving analysis results. Make sure a valid JavaScript Date is stored and displayed in a readable format like "Mar 18, 2026 — 5:30 AM".
2. Dark Mode doesn't apply to main content — Toggling dark mode only changes the sidebar. The main content area (analysis results, cards, score panels, upload area) stays white. Apply the dark theme to ALL components — backgrounds, cards, text, borders, score cards, memo banner, tables, and the upload area. If this is too complex to get right quickly, remove the dark mode toggle entirely.
3. No loading feedback during analysis — When a contract is being analyzed, it just shows "Analyzing..." with no progress indication. The AI analysis takes 20-30 seconds and users think it's frozen. Add staged loading messages that cycle through: "Reading contract..." → "Scoring risk dimensions..." → "Generating governance recommendation..." → "Building governance memo..." with a progress bar or spinner.
4. Cross-profile scores are nearly identical — The 4 organization profiles (Balanced, Regulated, Startup, Public) produce almost identical scores (e.g., 27 vs 27.5 vs 27.5 vs 27.5). The weight differences between profiles need to be more pronounced so users can clearly see how governance adapts to different organizational contexts. Increase the differentiation in the weight configurations so the scores visibly differ by at least 5-10 points across profiles.
5. Governance rationale contradicts approval level — The approval badge might say "Manager" but the rationale text says "VP-level approval". Make sure the rationale text always matches the displayed approval level in the executive summary. They should reference the same tier and the same approver roles.
6. "Clear All History" has no confirmation — Clicking "Clear All History" deletes everything immediately with no confirmation dialog. Add an "Are you sure? This cannot be undone." confirmation modal before clearing.
7. Rename "Analyse an Action" to "Analyse a Transaction" — Change this label everywhere it appears in the app (navigation, buttons, headings). "Action" is too vague.
